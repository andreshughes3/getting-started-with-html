The Basics of Web and HTML

What is the Web?
The web is a collection of html documents that essentially links everything together.

What are the major peieces of the Web?
(1) Computer running a browser that displays files found on the web
(2) Internet (basically a computer network)
--The browser will make requests via internet to servers using http
--http is a protocol that browsers uses to communicate with servers
(3) Servers: computers that host the files the make up the web

What is HTML?
HTML or hypertext markup language are documents that are made up of (1) text content, (2) markup, (3) references to other documents, (4) and links to other pages

What is a webpage?
a webpage is a text document written in a language called HTML. Web browser read these documents and then interpret and display them.

How does coding work?
text is written in a language that a computer can understand, but computers are stupid and can only read exact code.

Some Basic HTML Vocabulary

What is a Tag?
a tag is the name of an HTML element enclosed by brackets

What is an element?
an element is a single component of an html document and refers to everything within a set of opening and closing tags.

What is an attribute?
an attribute is a property of an html element and modifies the element

What is the difference between inline vs block elements
Block elements form a invisible box around the content inside them and will start a new line. Whereas, inline elements are just that "in line".
